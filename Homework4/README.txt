This is a basic NLP program that calculates the probability of Unigrams and Bigrams.
The program reads in 3 languages' corpus: English, Italian, and French.
The data is dumped into pickle file and then accuracy is calculated.

To run this program:
  Enter the name of the file that contains data. In this case, it is "data".
  The pickled files are added to the folder named "Pickle".
  
The program contains two files. One parses the data and stores it. The second file calculates accuracy using probability.
